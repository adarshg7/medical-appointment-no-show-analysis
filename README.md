Problem Statement : Analyze factors influencing missed medical appointments using statistical methods

Project Overview : Many patients misses scheduled medical appointment, leading to increase cost of expense
                   This project analyze factors influencing patients no-show 

Dataset : Kaggle: Medical Appointment No Shows (https://www.kaggle.com/datasets/joniarroba/noshowappointments)  
Size: 100,000+ appointments  
Key Features: PatientId, Gender, Age, ScheduledDay, AppointmentDay, SMS_received, Scholarship, etc.  
Target: No-show (Yes / No) 

Objectives
- Identify statistically significant factors that influence patient no-shows  
- Analyze the impact of waiting time, age, SMS reminders, and other variables  
- Translate statistical findings into actionable recommendations for healthcare management  

Key Insights & Recommendations
- **Reduce waiting time** for high-risk patients to decrease no-shows  
- **Reevaluate SMS reminder strategy**, as impact is statistically significant but practically small  
- Consider **age-based patient segmentation** for appointment management
